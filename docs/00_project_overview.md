# scAgent-DPM 项目概述

## 项目背景

单细胞转录组学（scRNA-seq）结合药物扰动实验为解析药物作用机制提供了前所未有的分辨率。然而，从原始测序数据到生物学解释的完整分析流程存在多重挑战：（1）质量控制参数选择高度依赖经验且缺乏系统性优化；（2）细胞类型注释的准确性和一致性受限于参考数据和注释方法；（3）药物扰动效应的量化缺乏整合多维度信息的统一评分框架；（4）扰动诱导的细胞状态动态转变难以用静态分析方法捕获。现有工具通常仅覆盖流程中的单一环节，缺乏端到端的自动化分析系统。

## 研究目标

构建 scAgent-DPM——一个面向药物干预机制解析的单细胞智能分析体，实现从数据摄入到自动报告生成的完整自动化分析流程。系统通过智能体规划器协调自适应质量控制、基础模型细胞注释、药物扰动响应评分（DPRS）和动态状态建模四个核心模块，并提供完整的可复现性保障机制。

## 系统定位

scAgent-DPM 定位为工具系统类研究，对标 Bioinformatics、Nucleic Acids Research Web Server Issue、Computers in Biology and Medicine 等期刊的工具论文标准。系统设计强调：

- **自动化**：从原始数据到分析报告的端到端自动化
- **可复现**：每一步的输入、参数、输出均被记录在 run manifest 中
- **模块化**：各分析模块可独立运行，支持灵活组合
- **可扩展**：通过配置接口支持不同注释方法和动态建模方法

## 与前三篇论文的关系

| 论文 | 方向 | 与本项目关系 |
|------|------|-------------|
| 第一篇 | 自适应单细胞质量控制 | QC 模块的核心方法基础 |
| 第二篇 | scGPT-KDMT 知识蒸馏多任务细胞注释 | 注释模块的高级方法接口 |
| 第三篇 | scLifeMamba / Mamba-LSTM 动态建模 | 动态建模模块的模型接口 |
| **第四篇（本项目）** | 药物干预机制解析智能分析体 | 集成前三篇成果，新增 DPRS 和 agent planner |

本项目不是前三篇工作的简单拼接，而是：
1. 提出了新的药物扰动响应评分（DPRS）框架
2. 设计了轻量级智能体规划器实现全流程自动化
3. 构建了完整的可复现性保障体系
4. 建立了统一的模块间接口和数据流规范

## 预期贡献

1. **方法贡献**：提出 DPRS 多维度药物扰动响应评分框架
2. **系统贡献**：构建首个集成自适应 QC、基础模型注释与动态建模的药物扰动分析自动化系统
3. **工程贡献**：提供可复现、可审计、可扩展的完整分析工程
4. **应用贡献**：为药物作用机制解析提供标准化分析工具
