# scAgent-DPM 投稿期刊分析

## 分析说明

本文为工具系统类论文，目标期刊为生物信息学/计算生物学领域的 SCI 期刊。以下分析基于期刊的发表范围、审稿周期、录用率和历史发表相似工具论文的情况。**影响因子和审稿周期为参考值，投稿前需再次核验最新数据。**

## 投稿策略建议

### 推荐投稿路线

1. **首选** → Bioinformatics (Original Paper 或 Application Note)
2. **备选 1** → Computers in Biology and Medicine
3. **备选 2** → BMC Bioinformatics
4. **补充备选** → Methods / Briefings in Functional Genomics

### 投稿前检查清单

- [ ] 确认目标期刊最新 IF、投稿格式要求、字数/页数限制
- [ ] 确认是否需要 Web 服务器（NAR Web Server Issue 必需）
- [ ] 确认代码开源许可证
- [ ] 确认数据可用性声明格式
- [ ] 准备 Cover letter
- [ ] 准备 Graphical abstract（如需要）
- [ ] 准备 Highlights（如需要）
- [ ] 检查参考文献格式是否符合目标期刊要求

### 时机选择

- Bioinformatics：准备好后随时投稿
- NAR Web Server Issue：注意每年 7 月的截止日期，需提前准备好 Web 服务器
- 其他期刊：滚动接受
